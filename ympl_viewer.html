What YMPL does

Converts messy process text ↔ structured YAML + SVG diagram. One portable file, works in any app.

API — embed in any app

// Browser <script src="ympl.js"></script> // + js-yaml CDN above it // Node.js const YMPL = require('./ympl.js') // Both environments — same call const result = YMPL.render(textOrYaml) result.yaml // YMPL 1.0 YAML string result.svg // SVG diagram string result.text // plain English description result.doc // { nodes, edges, ... } JS object
